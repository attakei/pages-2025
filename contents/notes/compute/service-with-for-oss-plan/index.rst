=================================
"For Open Source project"なプラン
=================================

Webサービスなどを利用していると、ごく稀に「無料プラン」「有料プラン」とは別に「OSS向けプラン」というものが存在する。
提供されているケースの多くでは、下記のような条件をもとに標準的な有料プランと同等の機能として提供してくれている。

* ある程度以上の期間メンテナンスされているOSSに関わっている。
* プランの利用自体が営利活動に該当しない。
* （場合によっては）プランを利用していることをOSSのドキュメントなどに記載する。

いわゆる商用利用に該当しない状態でOSSと関連する活動全般に使えるものもあるので、
委ねられている裁量はおそらくだいぶ広いはず。
このあたりは各サービスのポリシーに依存する話ではあるので、きちんと確認はしておく必要がある。

プラン提供側は「OSSエコシステムへの還元（意訳）」を題目に掲げているので、
悪用にならない範囲で互助だと思って使わせてもらうとよいだろう。

OSS向けプランが存在するもの
===========================

.. note:: 基本的に、類似サービス込みで探してみたものに限る。

* Netlify

  * `Open Source Plan <https://www.netlify.com/legal/open-source-policy/>`_ が存在する。
  * 今のところはGitHub PagesとCloudflareで事足りているので、利用していない。
  * 例えば、atsphinx全体とかをうまいこと統合管理出来るなら、試してみるのはありかも。

* JetBrains

  * IDE全般に `オープンソース向けライセンス <https://www.jetbrains.com/ja-jp/community/opensource/>`_ が存在する。
  * 個人ライセンスを持っているが、利用頻度が下がっているので一旦こっちに切り替えるか検討。

* DigitalOcean

  * `DigitalOcean Credits for Open Source <https://www.digitalocean.com/open-source>`_ が存在する。
  * ビルドが必要だったり、動的サイトを再管理したくなったときに検討したい。
